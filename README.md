EvanMi是我的个人域名「米表」
一个极简而精致的域名表，聚合了我手头所有有趣、有用、可售或待启的域名资产。
页面以深色/浅色双主题粒子背景为视觉核心，安静流动的连线粒子像域名之间若隐若现的关联，卡片式布局让域名一目了然，点击即可直达。
