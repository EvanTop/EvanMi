如果你手里有一批域名，想要一个页面：

- 看起来体面
- 别人一眼就能搜到、筛到
- 自己更新也方便

那 EvanMi 就是为这件事准备的。

---

## EvanMi 是什么？

**EvanMi 是一个轻量的开源域名米表**：

- **前台**：用卡片把域名展示出来，支持搜索、筛选后缀/注册商，打开即用
- **后台**：可以像管理表格一样维护域名，批量导入、导出、备份恢复

它的目标不是做“复杂系统”，而是做一个**简单、快速、长期好维护**的小工具。

---

## 你能用 EvanMi 做什么？

### 1）对外展示：更像作品集，不像表格

- 域名以卡片形式展示
- 搜索框一搜就定位
- 筛选注册商、筛选后缀，查找更快

适合发给朋友、客户，或者放到社交平台主页。

### 2）自己管理：更新域名列表更省心

- 新增 / 编辑 / 删除
- 批量操作
- **批量导入（覆盖式）**：导入新列表会直接覆盖旧列表，保证数据永远是最新的
- 导出 CSV 方便存档或发给别人

### 3）站点内容也能在后台改

很多米表会把“公告”和“外链”写死在页面里，改起来要改文件。

EvanMi 把这些也放进后台：

- **公告弹窗**：标题和内容随时改
- **Other Links**：可以添加链接（标题 / URL / 说明）

---

## 适合谁？

- 有域名要展示、要报价、要整理的人
- 不想折腾数据库、登录系统、复杂功能的人
- 想要“好看 + 好用 + 快”的人

---

## 一句话总结

**EvanMi = 一个好看的展示页 + 一个够用的管理后台。**

把你的域名清单变成一个随时可更新的米表，发出去也更专业。
